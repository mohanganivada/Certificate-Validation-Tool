The project is structured in such a way that executing main.py file will run  the whole project

Install virtual environment using the command "pip install virtualenv"

After install virtual environment use command outside the project folder i.e. Certificate Validation "virtualenv env"

A folder named "env" will be created

Be sure to activate the virtual environment by using the command "./evn/scripts/activate" in the same folder
as evn. To deactivate the env run command "deactivate"

Open config.py and add the relevant paths for excel and folder data. Run the file using command "python config.py" in the same directory

cd to Project folder i.e. Certificate Validation

Be sure to install the requirements.txt file (Command : pip install -r requirements.txt)
